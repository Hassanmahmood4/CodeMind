# CodeMind branding

Official logo assets live in `/public/codemind/`. Use the reusable `CodeMindLogo` component everywhere.

## Assets

| File | Use |
|------|-----|
| `codemind-logo-dark.svg` | Dark UI (bright logo) |
| `codemind-logo-light.svg` | Light backgrounds (dark logo) |
| `codemind-icon.svg` | Favicon / app icon |

Do not add duplicate logos elsewhere. All branding goes through `@/components/CodemindLogo`.
